# The Practitioner's Lexicon

An interactive knowledge base covering **1,000+ terms** across **20+ domains** — the shared language of strategy, delivery, product, data, and technology.

Live site: [https://piyushsingla.github.io/practioner_lexicon/](https://piyushsingla.github.io/practioner_lexicon/)

## Features

- **Search** — full-text search across terms, definitions, and abbreviations
- **Category filter** — filter by any of the 21 domains
- **A–Z browse** — jump to terms by first letter
- **Detail popup** — click any card to open a modal with the full definition, context, and related terms
- **Related terms** — navigate between linked terms directly from the modal
- **Bookmarks** — star terms to save them for quick reference

## Run locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## Deploy

```bash
npm run deploy
```

This builds the app and pushes it to the `gh-pages` branch automatically.
